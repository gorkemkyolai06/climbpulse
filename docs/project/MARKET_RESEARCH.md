# BayPulse Pazar Araştırması

## Pazar Büyüklüğü

- ABD'de ~6.500 indoor golf simulator tesisi (2025 tahmini)
- Bağımsız operatörlerin ~70%'i Excel, kağıt veya genel POS kullanıyor
- Ortalama tesis: 6-10 bay, yıllık gelir $400K-1.2M

## Hedef Segment

| Özellik | Profil |
|---------|--------|
| Bay sayısı | 4-16 |
| Lokasyon | Tek veya çift şube |
| Bölge | Illinois, Texas, Arizona, Colorado |
| Operatör | Aile işletmesi veya küçük franchise |

## Problem Doğrulama

- Bay durumu ve gelir kayıtları genellikle günlük Excel tablosunda
- Simulator arızaları WhatsApp/grup mesajında kayboluyor
- Bakım planları (kalibrasyon, turf) takvim uygulamasında dağınık
- Peak saat fiyatlandırması manuel hesaplanıyor

## Rakipler

| Rakip | Segment | Zayıf Nokta |
|-------|---------|-------------|
| GolfNow Pro | Büyük tesisler | Pahalı, karmaşık |
| EZLinks | Country club | Enterprise odaklı |
| TrackMan Dashboard | Simulator markası | Sadece TrackMan bayları |
| Excel/kağıt | Küçük operatörler | Ölçeklenemez, rapor yok |

## Farklılaşma

- Bay başına uygun fiyat ($5-12/bay/ay)
- Simulator markası bağımsız (TrackMan, Full Swing, Uneekor)
- Onarım + bakım + gelir tek platformda
- Premium Editorial Golf UX — operatör dostu, hızlı veri girişi
